Mosaic | Group Project

Matlab

February 2019 - March 2019

This project was done by me and another student, in the master course ‘Image Reproduction and Image Quality’. The purpose of the project was to reproduce an image using a database of 250 images. The images were chosen so the database would have a wide range of images. The goal was for the database to contain as many colours as possible. When an image is reproduced, the most suitable images are selected from the database based on the colour difference in the unit-independent colour system CIELAB. The reproduced image should resemble the original image as much as possible from a long distance, but the small images should be visible from a short distance. The smaller images that reproduce the original image, the better the detailed representation of the original image will be.
